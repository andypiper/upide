Erste Schritte mit μPIDE

μPIDE hat das Ziel, die Arbeit mit MicroPython-basierten Geräten so einfach wie möglich zu machen. Diese Hilfeseite soll durch die ersten Schritte führen.

Alle Daten sind dabei auf dem MicroPython-Gerät gespeichert. μPIDE erlaubt es, Python-Programme direkt auf dem Gerät zu erstellen, zu bearbeiten und auszuführen.

Die Dateiansicht

Die Dateisansicht links zeigt alles an, das auf dem MicroPythion-Gerät gespeichert ist. Das können Python-Dateien (auf .py endend) sowie andere Dateien sein. μPIDE erlaubt es in erster Linie, mit den Python-Dateien zu arbeiten. Diese Dateien können in Verzeichnissen organisiert sein.

Ein Doppelklick auf eine bestehende Datei in der Dateiansicht öffnet die entsprechende Datei. Wenn μPIDE die Bearbeitung des Dateiformats unterstützt, dann öffent sich ein Fenster, das die Bearbeitung der Datei erlaubt.

Ein Rechtsklick auf ein Objekt in der Dateiansicht öffent ein Kontextmenü, das es erlaubt, neue Dateien anzulegen, bestehende zu löschen, Dateien vom PC zu importieren, zu exportieren usw.

Der Editor

Wenn man eine bestehende Datei öffnet oder eine neue anlegt wird sich ein Editorfenster öffnen. Hier kann man die Datei nach Belieben verändern. Zwei Symbole erscheinen in der unteren Ecke des Editors: Ein schwarzes "Speichern"-Symbol und ein grünes "Ausführen"-Symbol.

Ein Klick auf das "Ausführen"-Symbol lädt den Code auf das Gerät und führt ihn aus. Das bedeutet nicht, dass die veränderten Daten auf dem Gerät gespeichert werden. Eine Datei im Editor zu verändern und auf dem Gerät auszuführen verändert keine Daten auf dem Gerät und die veränderte Datei wird nicht gespeichert.

Das Programm nutzt die Konsole am unteren Fensterrand der μPIDE, um Textdaten auszugeben und Eingaben engegenzunehmen. Auch Fehlermeldungen erscheinen hier.

Veränderte Daten werden mit dem "Speichern"-Symbol auf das Gerät zurückgeschrieben. Der Klick auf das Symbol schreibt die veränderten Daten in die bestehende Datei, der vorherige Inhalt geht dabei verloren. Die Änderungen werden so dauerhaft erhalten und bestehende Daten werden ersetzt.

Es lohnt sich, Änderungen regelmäßig zu speichern. Andernfalls können ungespeicherte Daten verloren gehen, wenn zum Beispiel das Gerät vom Computer getrennt wird.

Die REPL-Konsole

Einer der großen Vorteile von Python ist die Möglichkeit der interaktiven Anwendung. Die REPL-Komsole am unteren Fensterrand wird dafür genutzt.

Ein Klick auf das schwarze Symbol in der Konsole startet den interaktiven Modus (REPL steht für Read, Evaluate, Print und Loop) und erlaubt es direkt interaktiv mit dem Gerät zu kommunizieren.
